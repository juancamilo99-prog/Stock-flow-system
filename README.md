# 📦 StockFlow

> Sistema de gestión logística desarrollado para optimizar el control de inventario, entradas, salidas y trazabilidad de productos en un almacén.

---

## 📖 Sobre el proyecto

StockFlow es un proyecto personal cuyo objetivo es simular el desarrollo de un sistema de gestión logística inspirado en necesidades reales observadas durante mi experiencia profesional en el sector.

Más que un CRUD, pretende representar cómo se diseñaría una aplicación utilizada por operarios, encargados y coordinadores para gestionar el flujo de mercancías dentro de un almacén.

Este proyecto será desarrollado aplicando buenas prácticas de análisis, arquitectura y desarrollo backend con Java y Spring Boot.

---

## 🎯 Objetivos

- Gestionar productos e inventario.
- Registrar entradas y salidas de mercancía.
- Controlar el stock en tiempo real.
- Gestionar usuarios mediante roles.
- Mantener la trazabilidad de todos los movimientos.
- Diseñar una arquitectura escalable y mantenible.
